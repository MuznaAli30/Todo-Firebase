# Todo-Firebase ✅

## 📌 Description
A **Todo App** built with **React.js** and **Firebase** for real-time storage.  
Users can add, edit, and delete tasks with **real-time sync**.  

## 🚀 Features
- Add, edit, and delete tasks 📝  
- Firebase backend for real-time updates 🔄  
- Responsive UI design 🖥️📱  

## 🛠 Tech Stack
- React.js, Firebase, JavaScript 💻  

## ⚙️ Installation
1. Clone repo: `git clone https://github.com/MuznaAli30/Todo-Firebase.git`  
2. Install dependencies: `npm install`  
3. Start development server: `npm start`  
Open http://localhost:3000 in your browser 🌐
